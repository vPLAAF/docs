## **前言**

本标准作业程序（Standard Operating Procedures，下称 “本 SOP”）用于规范虚拟中国空军（vPLAAF）在 VATSIM
网络上的各项活动，包括组织管理、成员行为、飞行与任务的基本要求等，为全体成员提供统一的作业依据。

vPLAAF 为完全虚构的虚拟飞行组织，仅以飞行模拟与线上娱乐为目的。
vPLAAF 与现实的中国人民解放军空军、任何现实的政府机关、军事单位或商业机构**不存在任何形式的隶属、从属、合作或授权关系**。
本 SOP 所载内容不得用于任何现实航空活动或军事行为，不得作为现实飞行训练或作战的参考依据。

本 SOP 适用于：

- vPLAAF 全体成员
- 申请加入的成员
- 与 vPLAAF 开展协作的虚拟组织

本 SOP 应与以下文件一并适用：

- **VATSIM Code of Conduct（行为守则）**
- **VATSIM Code of Regulations**
- **VATSIM Special Operations Policy（特殊作业政策）**
- **vPLAAF 其他补充 SOP、训练大纲、任务简令、协定（LOA）等内部文件**

在 VATSIM 网络飞行时，如本 SOP 与 VATSIM 官方文件冲突，以 VATSIM 官方文件为最高优先。

3. 为训练、演练与任务组织提供制度基础。

2. 本 SOP 经 vPLAAF 司令批准后实施。
3. 如本 SOP 与以下文件不一致，以VATSIM 官方文件为准，并据此对本 SOP 进行相应修订：
    - VATSIM Code of Conduct
    - VATSIM Special Operations Policy
4. 如对条文存在疑义，由vPLAAF司令部作最终解释。

# **第二章 组织概况**

## **第五条 组织定位**

1. vPLAAF 为 VATSIM 网络上的虚拟军事航空组织，以模拟中国人民解放军空军风格的任务与训练为主要特色。
2. 基本原则：
    - 遵守 VATSIM 相关规则与政策；
    - 尊重现实航空安全原则，但不与现实军事或政治行为产生关联；
    - 提供沉浸式、秩序良好、具备学习与交流价值的飞行模拟环境。
3. vPLAAF 的全部活动均属虚拟环境之娱乐与兴趣交流，不具有任何现实军事意义。

------

## **第六条 主要任务类型**

vPLAAF 主要开展的虚拟任务类型包括但不限于：

1. 空防与截击任务模拟
2. 对地支援及对地攻击训练模拟（不涉及现实具体目标）
3. 空运、空投和补给任务模拟
4. 预警、侦察、指挥控制、空中护航、空中加油等特种任务模拟
5. 基础与进阶飞行训练（参考第二十六条）

# **第三章 成员管理**

## **第九条 加入条件**

申请加入 vPLAAF 的人员应满足下列基本条件：

1. 年龄21岁及以上
2. 拥有有效 VATSIM 账号，且无严重违规记录；
3. 在 VATSIM 拥有至少 500 小时的有效飞行时长，并具备成熟的飞行模拟经验，且愿意接受相应的专项飞行培训与考核；
4. 具备流利的英语和普通话陆空对话能力，能够理解并准确执行 ATC 指令；
5. 认同 vPLAAF 的定位、宗旨与规则，愿意遵守本 SOP 及其他内部规定；
6. 明确理解并接受：加入 vPLAAF 不赋予任何现实军人身份或相关权利。

------

## **第十条 申请流程**

1. 申请人通过 vPLAAF 官方网站或网站上的指定方式提交申请，内容包括但不限于：
    1. VATSIM ID 与使用昵称；
    2. 所使用的飞行模拟平台及常用机型；
    3. 过往虚拟飞行或管制相关经历（如有）；
    4. 加入 vPLAAF 的动机与期望。
2. 指定人员对申请进行初审，主要核查：
    1. VATSIM 账号状态及连线记录
    2. 申请信息是否完整、真实；
    3. 基本条件是否符合。
3. 安排语音或文字形式的交流，以进一步了解申请人的：
    1. 飞行经验与技术基础；
    2. 对 VATSIM 规则与 VSOA 概念的理解程度；
    3. 对 vPLAAF 风格与文化的适应度。
4. 审核结果：
    - 通过：申请人成为学员，并接收相应指引与训练安排；
    - 未通过：可由司令部视情况告知主要原因，并说明再次申请的时间间隔或条件。

## **第十二条 活跃度管理**

为保证 vPLAAF 作为 VSOA（Virtual Special Operations Association）成员的持续运作能力、审计合规性以及成员训练质量，依据《VATSIM
Special Operations Policy v4.0》相关规定，特制定以下活跃度管理条款。

**1. 活跃成员（Active Pilot）定义**

符合以下条件的成员被认定为 vPLAAF 活跃成员（Active Pilot）：

- **在过去 90 天内至少完成一次使用 vPLAAF 呼号的在线飞行**，并在飞行计划备注中标注组织网站。

- 完成组织要求的基础训练科目。

- 持续遵守 VATSIM CoC/CoR 及 vPLAAF 内部规章。

未满足第 1 条者自动视为非活跃成员（Inactive）。

**2. 组织最低活跃人数要求**

为满足 VSOP 合规性，vPLAAF 应保持：

> **至少 10 名活跃成员（Active Pilot）**

如活跃人数低于该标准，将被视为存在合规风险，并需立即采取招募与活跃激励措施。

**3. 最低在线飞行要求（组织层面）**

依据 VSOP 要求：

> **vPLAAF 必须在任意连续 60 天内完成至少 10 架次使用 vPLAAF 呼号的在线飞行。**

该要求为组织整体指标，不限定由特定成员完成，但需由 vPLAAF 各成员共同承担。

若未达到：

- 组织将被视为不满足 VSOA 活跃度审计标准。
- 在审计中可能被要求 60 天内补足 10 架次，否则可能失去资格。

**4. 个人活跃度要求**

为确保训练质量与组织战备能力，vPLAAF 规定成员：

- 每 90 天至少 1 次在线飞行。
- 每 180 天至少完成一次组织活动或内部训练。

**5. 非活跃处理机制**

若成员超过 90 天无在线飞行：

1. 状态由 Active 自动转为 **Inactive**。
2. 暂停训练权限、暂停组织任务资格。
3. 超过 6 个月未恢复活动者，可由人员部门执行：
    - 清理出飞行员名册（Roster），但可保留历史档案。
    - 若返回组织，可申请重新激活并完成必要的补课训练。

**6. 强制活跃度审查（Audit Readiness）**

为满足年审（Audit）时的数据提交要求，组织必须确保：

- 所有飞行记录可查、可验证。
- 所有 Active Pilot 名单公开可查（至少显示 CID）。

任何成员若拒绝遵守飞行记录提交、信息更新要求，可被暂时禁止参与任务。

**7. 年度活动义务**

依据 VSOP：

> **vPLAAF 必须每年举办至少 1 次公开活动（Event）并在 VATSIM 网络上进行。**

活动须：

- 对外公告（VATSIM、论坛、社交媒体）。
- 形成活动报告（Post-Event Report）。

参与年度活动视为一次合格的活跃度贡献。

**8. 活跃度激励与奖励**

为鼓励成员活跃飞行，vPLAAF设立以下激励：

- 月度/季度 飞行小时/次数 Top 10 公示
- 内部晋升参考指标

**9. 特殊情况例外**

成员如因以下情况无法满足活跃度要求，可申请豁免：

- 服役、长期出差等个人事由
- 医疗相关情况
- VATSIM 账号技术问题

需通过人员部门审批并登记，最长可豁免 180 天。

# **第五章 成员行为规范**

## **第十七条 基本行为要求**

1. 全体成员在 VATSIM 网络上须保持专业、礼貌、尊重的态度，遵守 VATSIM Code of Conduct。
2. 成员不得以“军机”身份向任何 ATC 或其他飞行员提出不合理的优先权或特权要求。
3. 成员不得贬低、嘲讽或恶意干扰其他 VATSIM 用户，包括飞行员与管制员。
4. 在任何公开频道（包括频率、文字频道及社交平台）表现出的行为均会影响 vPLAAF 形象，成员应自觉维护组织声誉。

------

## **第十八条 无线电与通话规范**

1. 成员应使用正确的呼号、频率和通话格式。
2. vPLAAF的呼号具有固定格式:
    - **作战单位**
      （战斗机、直升机、特种机等），呼号为参考[维基百科](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%9B%BD%E4%BA%BA%E6%B0%91%E8%A7%A3%E6%94%BE%E5%86%9B%E7%A9%BA%E5%86%9B%E4%B8%8E%E6%B5%B7%E5%86%9B%E8%88%AA%E7%A9%BA%E5%85%B5%E7%BC%96%E5%88%B6%E5%BA%8F%E5%88%97)
      的**五位数字战术编号**（例如：在路桥机场训练时的歼-10A呼号80341），**该类单位在未经授权或未签署LOA的情况下不得离开中国大陆空域
      **
    - **运输单位**（客机、货机、公务机等），呼号前缀为CKJ，中文呼号“军航”，英文呼号“China Air Force”，**该类单位可执行国内或国际运输任务
      **
3. 不得在频率进行与飞行无关的交谈，防止妨碍他人通信。
4. 若无法理解或执行 ATC 指令，应主动说明情况，并请求更简单或更适合自身能力的指令。
5. 遇到操作失误或连接问题，应坦诚告知，并配合 ATC 或其他成员进行调整。

------

## **第十九条 内部交流与公共形象**

1. 在 vPLAAF 内部通讯平台（如 Discord 等）中：
    1. 禁止发表侮辱、人身攻击、恶意挑衅等言论；
    2. 不得涉及现实政治、种族、宗教等高度敏感或争议话题。
2. 成员对外提及 vPLAAF 时，不得以组织名义发表不当或与本组织精神相悖的言论。

------

## **第二十条 违规处理**

1. 若成员违反本 SOP、VATSIM 规则或其他相关规定，司令部可视情节轻重采取下列措施之一或组合：
    1. 口头或书面提醒；
    2. 要求参加额外训练；
    3. 暂停参与部分或全部 vPLAAF 活动的资格；
    4. 撤销职务；
    5. 取消成员资格；
    6. 对涉及 VATSIM 违规的情况，按规定向相关部门报告。
2. 处理决定应在合理范围内向当事人说明理由，并做好记录。
3. 成员如对处理结果有异议，可在规定时间内提出书面申诉，由司令部或指定复议小组进行复核并作出最终决定。

# **第六章 飞行与任务一般要求**

## **第二十一条 任务类别**

- vPLAAF 的飞行任务一般分为：
    1. 基础训练任务；
    2. 常规科目任务；
    3. 联合或演练任务；
    4. 军事活动；
    5. 其他经司令部批准的任务类型。
- 各类任务应指定任务负责人或任务指挥，负责：
    1. 任务策划与飞行程序拟定；
    2. 任务简报与风险提示；
    3. 任务结束后的总结与反馈收集。

------

## **第二十二条 飞行计划与呼号**

1. 所有以 vPLAAF 名义在 VATSIM 进行的飞行，应在连接前提交 Flight Plan，具体要求包括：

    1. 正确填写机型 ICAO 代号；

    2. 使用 vPLAAF 统一规定的呼号格式；

    3. 在备注栏（RMK/）中标注 vPLAAF 身份及任务类型，例如：

       ```
       RMK/VSO - vPLAAF - FORMATION TRAINING – WWW.VPLAAF.ORG
       RMK/VSO - vPLAAF – INTERCEPT TRAINING – WWW.VPLAAF.ORG
       ```

2. 呼号设计应在不与其他组织冲突的前提下保持清晰、一致，具体格式由任务简令及相关 LOA 详细规定。

------

## **第二十三条 机型与替代**

1. 成员应尽量使用与模拟的现实机型性能和类别相近的模拟机型。
2. 对于 VATSIM 机型数据库中不存在的机型，应由 vPLAAF 统一指定替代 ICAO 代号。
3. 成员在选择机型时，应确保自身对该机型的基本飞行操作足够熟悉，以确保任务顺利和网络安全。

------

## **第二十四条 飞行记录**

- vPLAAF 应通过官方网站、ACARS 系统或其他方式记录成员飞行数据，包括：
    1. 日期与任务类型；
    2. 起降机场与飞行时长；
    3. 参与成员名单；
    4. 其他必要信息。
- 飞行记录可用于：
    1. 统计飞行小时数；
    2. 评估训练效果与任务完成情况；
    3. 作为授予等级、勋章或奖励的依据。

# **第七章 训练与资格**

------

## **第二十五条 目标**

- 确保成员能安全规范飞行
- 掌握所需技能
- 熟练执行 SOP

------

## **第二十六条 训练内容**

1. **VFR基础（不少于1小时）**
   
    - **训练目标**
    
      完成本阶段训练后，学员应能够：
    
      1. 理解影响飞机爬升、下降与能量管理的主要因素；
      2. 在目视条件下，按照标准程序建立并保持稳定的本场飞行；
      3. 正确执行起飞、初始爬升、平飞、下降、进近、复飞等阶段的操作流程；
      4. 在本场飞行中，能够根据需要选择并保持不同空速，且飞行姿态与配平合理；
      5. 在 ATC 存在或无 ATC 情况下，正确进行陆空通话或 CTAF 通话。
    
    - **主要内容**
        1. **地面与起飞阶段**
           - 起飞前检查单的规范使用；
           - 标准起飞程序与初始爬升；
           - 侧风起飞的基本修正方法（如适用）。
        2. **本场飞行（Circuits）**
           - 标准本场高度（通常 1000 ft AGL，按机场规定）；
           - 上风边、侧风边、顺风边、底边、最后进近的建立与保持；
           - 本场内爬升、下降阶段的姿态、功率与配平控制；
           - 不同空速下的爬升与下降控制；
           - 无襟翼 / 少襟翼本场（如机型适用）。
        3. **进近、复飞与异常情况**
           - 稳定进近的建立；
           - 标准复飞程序；
           - 模拟低高度复飞或不稳定进近中止；
           - 基本噪音控制与航迹管理意识。
        4. **通信与程序**
           - 受控机场下遵循 ATC 指令完成本场飞行；
           - 无 ATC 情况下的标准 CTAF 通话流程；
           - 正确理解并执行进近、起飞与本场相关指令。
        
    - **阶段要求**
      学员应在教官监督下完成不少于 1 小时的 VFR 基础飞行，并达到教官认可的最低操纵与程序标准，方可进入下一阶段。
      
    - **阶段考核标准**
    
      - 学员应能够在教官监督下，达到以下最低标准：
      - 本场高度保持在 **±100 ft** 以内；
      - 空速控制在目标值 **±10 KIAS** 以内；
      - 航向保持误差不超过 **±5°**；
      - 姿态控制平稳，无明显过度操纵；
      - 能在爬升、下降结束后，恢复稳定的平飞状态；
      - 正确使用配平，避免长期操纵力；
      - 能独立、安全完成至少 **3 个完整 VFR 本场**。
    
2. **VFR导航与夜航（不少于2小时）**
   
    - **训练目标**
    
      完成本阶段训练后，学员应能够：
    
      1. 在目视飞行条件下，完成短程航线的规划、执行与返场；
      2. 理解并正确使用基础导航手段进行航迹保持与定位；
      3. 在日间与夜间条件下，保持对高度、航向、空速与态势的有效控制；
      4. 在目视飞行中具备基本的地形、障碍物与交通避让意识；
      5. 在 ATC 存在或无 ATC 情况下，正确完成相关通话与程序。
    
    - **主要内容**
        1. **航前准备与航线规划**
           - 简单 VFR 航线规划（起降机场选择、备降考虑）；
           - 航向、距离、时间与油量的基础计算；
           - 飞行计划（Flight Plan）的基本填写与提交。
        2. **目视导航飞行**
           - 通过地标、航迹与目视参考进行导航；
           - 使用基础导航设备（如 VOR / DME / GPS 等，视机型与模拟条件）；
           - 航线飞行中的高度、航向与空速保持；
           - 航线偏差的识别与修正。
        3. **目视飞行中的态势感知**
           - 地形与障碍物避让原则；
           - 空域边界与受限空域的识别；
           - 交通观察与避让意识（“See and Avoid”）。
        4. **夜航基础**
           - 夜间视觉特性与空间定向风险；
           - 夜间起飞、爬升、进近与着陆的注意事项；
           - 机场灯光、跑道灯光与目视助航灯的识别与使用；
           - 夜间目视导航的限制与风险意识。
        5. **通信与程序**
           - 航线飞行中的 ATC 通话流程；
           - 无 ATC 情况下的标准通话与位置报告；
           - 正确理解并执行高度、航向与航线相关指令。
        
    - **阶段要求**
      学员应完成不少于 2 小时的导航训练（含日间与夜间科目），能够在教官提供一般性引导的前提下，独立完成一段简单航线并安全返场。
      
    - **阶段考核标准**
    
      学员应能够在教官监督下，达到以下最低标准：
    
      - 能正确识别并使用至少一种基础导航台或导航手段（如 VOR / DME / GPS），用于航迹保持与位置确认；
      - 能正确识别并调谐指定的 VOR / DME 导航台，并理解其径向与距离信息；
      - 能建立并保持指定的 VOR 径向或 VOR-DME 弧线，航迹偏差可控且修正及时；
      - 能通过目视参考或基础导航手段维持合理航迹；
      - 能通过导航台信息与目视参考相结合，维持合理的态势感知与航迹判断；
      - 夜航阶段操作平稳，无明显空间定向错误；
      - 能安全、独立完成至少一次完整 VFR 航线飞行（含起飞与着陆）。
    
3. **IFR 程序与仪表飞行训练（不少于 3 小时）**
   
    - **训练目标**
    
      完成本阶段训练后，学员应能够：
    
      1. 在仪表气象条件下，依靠仪表安全控制飞机姿态与飞行轨迹；
      2. 理解并正确执行标准仪表飞行程序，包括起飞、离场与进场程序、航路、进近与复飞；
      3. 正确使用常见仪表进近与导航手段完成受控飞行；
      4. 在 ATC 指挥下，准确理解并执行高度、航向、速度与航路相关指令；
      5. 在高工作负荷环境下，保持基本的态势感知与程序一致性
    
    - **主要内容**
        1. **仪表飞行基础**
           - 基本飞行仪表的识别与使用；
           - 姿态、功率与性能的仪表化控制；
           - 基于仪表的爬升、平飞与下降；
           - 标准转弯、改平与航向保持。
        2. **标准仪表程序（SID / STAR）**
           - 标准仪表起飞程序（SID）的理解与执行；
           - 标准仪表进场程序（STAR）的理解与执行；
           - 高度、速度限制的识别与遵守；
           - 程序偏差的识别与修正。
        3. **航路与导航**
           - 基于 VOR / DME / RNAV 的航路飞行；
           - 航路段高度管理与航迹保持；
           - 航路中 ATC 指令的执行（雷达引导、直飞、改航、偏置等）；
           - 航路态势感知与工作负荷管理。
        4. **仪表进近**
           - 精密进近与非精密进近的基本概念；
           - ILS / LOC / VOR / NDB / RNAV（LNAV）等进近程序的执行；
           - 进近稳定性判断与持续监控；
           - 最低高度 / 决断高度的理解与遵守。
        5. **等待与复飞**
           - 等待航线（Holding）的建立、进入与保持；
           - 标准复飞程序的理解与执行；
           - 不稳定进近的中止与改飞；
           - 复飞后程序与 ATC 协调。
        6. **夜间 IFR 与复杂情景**
           - 夜间 IFR 起飞与进近注意事项；
           - 多重指令下的程序执行；
           - 模拟设备或系统异常下的基本处置（在教官监控下）。
        
    - **阶段要求**
      学员应在线完成不少于 3 小时的 IFR 训练，并在至少一次完整 IFR 航线中，展示对基础程序和 ATC指令的正确理解与执行，经教官评定合格后方可进入第四阶段。
      
    - **阶段考核标准**
    
      学员应能够在教官监督下，达到以下最低标准：
    
      - 能正确理解并执行指定的 SID、STAR、航路及进近程序，遵守相关高度、速度与航迹限制；
      - 能使用 VOR / DME / RNAV 等仪表导航手段保持指定航迹，并进行合理、及时的修正；
      - 能独立完成至少一种仪表进近（ILS / LOC / VOR / RNAV），并正确理解与遵守 MDA / DA；
      - 能在需要时立即、正确执行标准复飞程序，并完成复飞后的程序衔接；
      - 能建立并保持等待航线，理解进入方式并保持航迹稳定；
      - 能准确理解、复诵并执行 ATC 的高度、航向、速度及程序指令；
      - 能在较高工作负荷环境下维持程序优先级判断，避免明显程序性错误；
      - 能安全、完整地完成至少一次 IFR 航线飞行（起飞—航路—进近—着陆或复飞）。
    
4. **特种作业与机型专项训练**

- **训练目标**

  完成本阶段训练后，飞行员应能够：

  1. 理解并执行 vPLAAF 虚拟特种作业相关的基本战术与程序；
  2. 熟练掌握所飞行机型的专项操作流程与性能特性；
  3. 在符合 VATSIM 与 VSOP 规则的前提下，安全执行低空、编队及特定任务科目；
  4. 在多机协同或复杂任务环境下，保持程序一致性、通信规范与态势感知；
  5. 具备参与 vPLAAF 正式任务或联合演练的基本能力

- **主要内容**（可根据机型与任务类型选取若干科目组合完成）
    1. **机型专项操作**
       - 机型启动、关车与特殊系统操作流程；
       - 机型性能限制、包线理解与能量管理；
       - 常见异常与应急情况的模拟处置（如发动机异常、系统失效等）。
    2. **低空飞行与航迹管理**
       - 低空飞行的高度、速度与风险控制原则；
       - 地形跟随与地形规避意识；
       - 在允许空域内进行的低空航迹规划与保持；
       - 低空飞行中与 ATC 或任务指挥的协调。
    3. **编队飞行**
       - 基本编队队形（如双机、三机或四机）；
       - 编队集合与解散程序；
       - 编队内位置保持、相对运动与安全间隔；
       - 编队飞行中的通信分工与职责划分。
    4. **特种任务科目**
       - 空运、空投与补给任务流程；
       - 护航、伴飞与编组转场任务；
       - 任务空域内的航迹保持与冲突规避；
       - 任务中与其他虚拟组织的协同（如适用）。
    5. **高级科目（视条件选取）**
       - 空中加油程序；
       - 基础空战机动（能量管理、进攻/防御机动等，仅限允许空域）；
       - 特殊使用空域（SUA / MOA / Military Training Routes）的理解与模拟使用。
    6. **任务流程与风险管理**
       - 任务前简报（Briefing）的参与与理解；
       - 任务执行中的程序遵循与变更应对；
       - 任务后讲评（Debriefing）与经验总结；
       - 网络安全、他人体验与合规意识。

- **阶段要求**
    1. 飞行员应完成不少于 4 小时的高级特种作业训练
    2. 通过由教官或标准与评估组组织的综合性实践考试，验证其在任务规划、程序执行、通信和飞行安全等方面达到 vPLAAF 规定标准。
    3. 通过本阶段考核后，飞行员方可被认定为具备参加 vPLAAF 组织的正式任务/联合演练之资格。

## **第二十七条 资格评定与保持**

1. 各训练阶段结束后，需通过理论测试、模拟任务及实践考核等方式进行评定。
2. 成员取得相应训练等级或资格后，如长期未参加相关类型飞行，司令部可要求其进行复训或再评估。
3. 具体评定标准、考核方式和资格保持期限以训练部为准。
