## **前言**

本标准作业程序（Standard Operating Procedures，下称 “本 SOP”）用于规范虚拟中国空军（vPLAAF）在 VATSIM 网络上的各项活动，包括组织管理、成员行为、飞行与任务的基本要求等，为全体成员提供统一的作业依据。

vPLAAF 为完全虚构的虚拟飞行组织，仅以飞行模拟与线上娱乐为目的。
 vPLAAF 与现实的中国人民解放军空军、任何现实的政府机关、军事单位或商业机构**不存在任何形式的隶属、从属、合作或授权关系**。
 本 SOP 所载内容不得用于任何现实航空活动或军事行为，不得作为现实飞行训练或作战的参考依据。

本 SOP 适用于：

- vPLAAF 全体成员
- 申请加入的成员
- 与 vPLAAF 开展协作的虚拟组织

本 SOP 应与以下文件一并适用：

- **VATSIM Code of Conduct（行为守则）**
- **VATSIM Code of Regulations**
- **VATSIM Special Operations Policy（特殊作业政策）**
- **vPLAAF 其他补充 SOP、训练大纲、任务简令、协定（LOA）等内部文件**

在 VATSIM 网络飞行时，如本 SOP 与 VATSIM 官方文件冲突，以 VATSIM 官方文件为最高优先。

# **第二章 组织概况**

## **第五条 组织定位**

1. vPLAAF 为 VATSIM 网络上的虚拟军事航空组织，以模拟中国人民解放军空军风格的任务与训练为主要特色。
2. 基本原则：
   - 遵守 VATSIM 相关规则与政策；
   - 尊重现实航空安全原则，但不与现实军事或政治行为产生关联；
   - 提供沉浸式、秩序良好、具备学习与交流价值的飞行模拟环境。
3. vPLAAF 的全部活动均属虚拟环境之娱乐与兴趣交流，不具有任何现实军事意义。

------

## **第六条 主要任务类型**

vPLAAF 主要开展的虚拟任务类型包括但不限于：

1. 空防与截击任务模拟
2. 对地支援及对地攻击训练模拟（不涉及现实具体目标）
3. 空运、空投和补给任务模拟
4. 预警、侦察、指挥控制、空中护航、空中加油等特种任务模拟
5. 基础与进阶飞行训练（参考第二十六条）

# **第三章 成员管理**

## **第九条 加入条件**

申请加入 vPLAAF 的人员应满足下列基本条件：

1. 年龄21岁及以上
2. 拥有有效 VATSIM 账号，且无严重违规记录；
3. 在VATSIM具备熟练的飞行模拟经验，并且愿意接受相应特殊飞行培训和考核；
4. 具备流利的英语和普通话陆空对话能力，能够理解并准确执行 ATC 指令；
5. 认同 vPLAAF 的定位、宗旨与规则，愿意遵守本 SOP 及其他内部规定；
6. 明确理解并接受：加入 vPLAAF 不赋予任何现实军人身份或相关权利。

------

## **第十条 申请流程**

1. 申请人通过 vPLAAF 官方网站或网站上的指定方式提交申请，内容包括但不限于：
   1. VATSIM ID 与使用昵称；
   2. 所使用的飞行模拟平台及常用机型；
   3. 过往虚拟飞行或管制相关经历（如有）；
   4. 加入 vPLAAF 的动机与期望。
2. 指定人员对申请进行初审，主要核查：
   1. VATSIM 账号状态及连线记录
   2. 申请信息是否完整、真实；
   3. 基本条件是否符合。
3. 安排语音或文字形式的交流，以进一步了解申请人的：
   1. 飞行经验与技术基础；
   2. 对 VATSIM 规则与 VSOA 概念的理解程度；
   3. 对 vPLAAF 风格与文化的适应度。
4. 审核结果：
   - 通过：申请人成为学员，并接收相应指引与训练安排；
   - 未通过：可由司令部视情况告知主要原因，并说明再次申请的时间间隔或条件。

## **第十二条 活跃度管理**

为保证 vPLAAF 作为 VSOA（Virtual Special Operations Association）成员的持续运作能力、审计合规性以及成员训练质量，依据《VATSIM Special Operations Policy v4.0》相关规定，特制定以下活跃度管理条款。

**1. 活跃成员（Active Pilot）定义**

符合以下条件的成员被认定为 vPLAAF 活跃成员（Active Pilot）：

- **在过去 90 天内至少完成一次使用 vPLAAF 呼号的在线飞行**，并在飞行计划备注中标注组织网站。

- 完成组织要求的基础训练科目。

- 持续遵守 VATSIM CoC/CoR 及 vPLAAF 内部规章。

未满足第 1 条者自动视为非活跃成员（Inactive）。

**2. 组织最低活跃人数要求**

为满足 VSOP 合规性，vPLAAF 应保持：

> **至少 10 名活跃成员（Active Pilot）**

如活跃人数低于该标准，将被视为存在合规风险，并需立即采取招募与活跃激励措施。

**3. 最低在线飞行要求（组织层面）**

依据 VSOP 要求：

> **vPLAAF 必须在任意连续 60 天内完成至少 10 架次使用 vPLAAF 呼号的在线飞行。**

该要求为组织整体指标，不限定由特定成员完成，但需由 vPLAAF 各成员共同承担。

若未达到：

- 组织将被视为不满足 VSOA 活跃度审计标准。
- 在审计中可能被要求 60 天内补足 10 架次，否则可能失去资格。

**4. 个人活跃度要求**

为确保训练质量与组织战备能力，vPLAAF 规定成员：

- 每 90 天至少 1 次在线飞行。
- 每 180 天至少完成一次组织活动或内部训练。

**5. 非活跃处理机制**

若成员超过 90 天无在线飞行：

1. 状态由 Active 自动转为 **Inactive**。
2. 暂停训练权限、暂停组织任务资格。
3. 超过 6 个月未恢复活动者，可由人员部门执行：
   - 清理出飞行员名册（Roster），但可保留历史档案。
   - 若返回组织，可申请重新激活并完成必要的补课训练。

**6. 强制活跃度审查（Audit Readiness）**

为满足年审（Audit）时的数据提交要求，组织必须确保：

- 所有飞行记录可查、可验证。
- 所有 Active Pilot 名单公开可查（至少显示 CID）。

任何成员若拒绝遵守飞行记录提交、信息更新要求，可被暂时禁止参与任务。

**7. 年度活动义务**

依据 VSOP：

> **vPLAAF 必须每年举办至少 1 次公开活动（Event）并在 VATSIM 网络上进行。**

活动须：

- 对外公告（VATSIM、论坛、社交媒体）。
- 形成活动报告（Post-Event Report）。

参与年度活动视为一次合格的活跃度贡献。

**8. 活跃度激励与奖励**

为鼓励成员活跃飞行，vPLAAF设立以下激励：

- 月度/季度 飞行小时/次数 Top 10 公示
- 内部晋升参考指标

**9. 特殊情况例外**

成员如因以下情况无法满足活跃度要求，可申请豁免：

- 服役、长期出差等个人事由
- 医疗相关情况
- VATSIM 账号技术问题

需通过人员部门审批并登记，最长可豁免 180 天。

# **第五章 成员行为规范**

## **第十七条 基本行为要求**

1. 全体成员在 VATSIM 网络上须保持专业、礼貌、尊重的态度，遵守 VATSIM Code of Conduct。
2. 成员不得以“军机”身份向任何 ATC 或其他飞行员提出不合理的优先权或特权要求。
3. 成员不得贬低、嘲讽或恶意干扰其他 VATSIM 用户，包括飞行员与管制员。
4. 在任何公开频道（包括频率、文字频道及社交平台）表现出的行为均会影响 vPLAAF 形象，成员应自觉维护组织声誉。

------

## **第十八条 无线电与通话规范**

1. 成员应使用正确的呼号、频率和通话格式。
2. 不得在频率进行与飞行无关的交谈，防止妨碍他人通信。
3. 若无法理解或执行 ATC 指令，应主动说明情况，并请求更简单或更适合自身能力的指令。
4. 遇到操作失误或连接问题，应坦诚告知，并配合 ATC 或其他成员进行调整。

------

## **第十九条 内部交流与公共形象**

1. 在 vPLAAF 内部通讯平台（如 Discord 等）中：
   1. 禁止发表侮辱、人身攻击、恶意挑衅等言论；
   2. 不得涉及现实政治、种族、宗教等高度敏感或争议话题。
2. 成员对外提及 vPLAAF 时，不得以组织名义发表不当或与本组织精神相悖的言论。

------

## **第二十条 违规处理**

1. 若成员违反本 SOP、VATSIM 规则或其他相关规定，司令部可视情节轻重采取下列措施之一或组合：
   1. 口头或书面提醒；
   2. 要求参加额外训练；
   3. 暂停参与部分或全部 vPLAAF 活动的资格；
   4. 撤销职务；
   5. 取消成员资格；
   6. 对涉及 VATSIM 违规的情况，按规定向相关部门报告。
2. 处理决定应在合理范围内向当事人说明理由，并做好记录。
3. 成员如对处理结果有异议，可在规定时间内提出书面申诉，由司令部或指定复议小组进行复核并作出最终决定。

# **第六章 飞行与任务一般要求**

## **第二十一条 任务类别**

- vPLAAF 的飞行任务一般分为：
  1. 基础训练任务；
  2. 常规科目任务；
  3. 联合或演练任务；
  4. 军事活动；
  5. 其他经司令部批准的任务类型。
- 各类任务应指定任务负责人或任务指挥，负责：
  1. 任务策划与飞行程序拟定；
  2. 任务简报与风险提示；
  3. 任务结束后的总结与反馈收集。

------

## **第二十二条 飞行计划与呼号**

1. 所有以 vPLAAF 名义在 VATSIM 进行的飞行，应在连接前提交 Flight Plan，具体要求包括：

   1. 正确填写机型 ICAO 代号；

   2. 使用 vPLAAF 统一规定的呼号格式；

   3. 在备注栏（RMK/）中标注 vPLAAF 身份及任务类型，例如：

      ```
      RMK/VSO - vPLAAF - FORMATION TRAINING – WWW.VPLAAF.ORG
      RMK/VSO - vPLAAF – INTERCEPT TRAINING – WWW.VPLAAF.ORG
      ```

2. 呼号设计应在不与其他组织冲突的前提下保持清晰、一致，具体格式由任务简令及相关 LOA 详细规定。

------

## **第二十三条 机型与替代**

1. 成员应尽量使用与模拟的现实机型性能和类别相近的模拟机型。
2. 对于 VATSIM 机型数据库中不存在的机型，应由 vPLAAF 统一指定替代 ICAO 代号。
3. 成员在选择机型时，应确保自身对该机型的基本飞行操作足够熟悉，以确保任务顺利和网络安全。

------

## **第二十四条 飞行记录**

- vPLAAF 应通过官方网站、ACARS 系统或其他方式记录成员飞行数据，包括：
  1. 日期与任务类型；
  2. 起降机场与飞行时长；
  3. 参与成员名单；
  4. 其他必要信息。
- 飞行记录可用于：
  1. 统计飞行小时数；
  2. 评估训练效果与任务完成情况；
  3. 作为授予等级、勋章或奖励的依据。

------

# **第七章 训练与资格**

------

## **第二十五条 目标**

- 确保成员能安全规范飞行
- 掌握所需技能
- 熟练执行 SOP

------

## **第二十六条 训练内容**

1. **VFR基础（不少于1小时）**
   - **训练目标**
      使学员在良好气象条件下熟悉所飞机型的基本操纵特性与 VATSIM 环境下的基本程序，能够在教官协助下安全完成一次标准飞行。
   - **主要内容**
     1. 地面滑行、转弯、进出滑行道等地面操纵；
     2. 起飞、爬升、平飞、下降、进近与着陆等基本动作；
     3. 本场飞行、连续起落及复飞；
     4. 与 ATC 的基础陆空通话流程（放行、进出跑道、起飞、进近、落地等）；
     5. 基本飞行安全意识与紧急中止起飞、复飞的原则。
   - **阶段要求**
      学员应在教官监督下完成不少于 1 小时的 VFR 基础飞行，并达到教官认可的最低操纵与程序标准，方可进入下一阶段。
2. **VFR导航与夜航（不少于2小时）**
   - **训练目标**
      使学员掌握在 VFR 条件下进行短途导航、航线飞行与基础夜航的能力，具备独立完成简单任务的导航基础。
   - **主要内容**
     1. VOR / NDB / DME / GPS 等基础导航设备的使用；
     2. 航线规划、航向/高度/油量计算与飞行计划基础填写；
     3. 通过地标、导航台和航径进行导航；
     4. 目视飞行下的地形与交通避让原则；
     5. 白天与夜间的目视航线飞行与进近；
     6. 模拟失速、恢复等基本异常情况处置（在教官监控下进行）。
   - **阶段要求**
      学员应完成不少于 2 小时的导航训练（含日间与夜间科目），能够在教官提供一般性引导的前提下，独立完成一段简单航线并安全返场。
3. **IFR 程序与仪表飞行训练（不少于 3 小时）**
   - **训练目标**
      使学员在 IMC / IFR 条件下，掌握基本仪表飞行技术与标准程序，能够在受控空域内遵照 ATC 指令完成航线及进近任务。
   - **主要内容**
     1. 基本仪表使用；
     2. 标准航路（SIDs / STARs）执行与高度、速度限制遵守；
     3. 精密与非精密进近程序（ILS / LOC / VOR / NDB / RNAV 等）；
     4. 等待航线的建立、进入与保持；
     5. 标准复飞程序的执行；
     6. 至少一次中短程 IFR 航线飞行，包括起飞、巡航、进近与落地；
     7. 夜间 IFR 起降与灯光。
   - **阶段要求**
      学员应在线完成不少于 3 小时的 IFR 训练，并在至少一次完整 IFR 航线中，展示对基础程序和 ATC 指令的正确理解与执行，经教官评定合格后方可进入第四阶段。
4.  **特种作业与机型专项训练**
   - **训练目标**
      使飞行员掌握 vPLAAF 作为虚拟特殊作业组织所需的核心军事科目，包括低空、编队、空战机动、空中加油等，在保证网络安全与他人体验的前提下，具备参与特种任务的能力。
   - **主要内容**（可根据机型与任务类型选取若干科目组合完成）
     1. 机型专用操作与异常情况处置（如单发、发动机故障后迫降程序的模拟）；
     2. 低空飞行与地形跟随（在符合 VSOA 和 ATC 要求的空域与高度内进行）；
     3. 编队飞行：队形保持、进出队形与 Join-up / Break 机动；
     4. 空中加油：预接近、接近、保持与脱离流程（如模拟设备支持）；
     5. 空投、空运与补给任务；
     6. 基本空战机动：能量管理、进攻/防御转弯等基础动作（在允许空域内进行，且不干扰第三方流量）；
     7. 特殊使用空域（SUA）、MOA、Military Training Routes 等的程序理解与模拟使用；
     8. 风险评估与任务前后简报/讲评流程。
   - **阶段要求**
     1. 飞行员应完成不少于 4 小时的高级特种作业训练
     2. 通过由教官或标准与评估组组织的综合性实践考试，验证其在任务规划、程序执行、通信和飞行安全等方面达到 vPLAAF 规定标准。
     3. 通过本阶段考核后，飞行员方可被认定为具备参加 vPLAAF 组织的正式任务/联合演练之资格。

------

## **第二十七条 资格评定与保持**

1. 各训练阶段结束后，需通过理论测试、模拟任务及实践考核等方式进行评定。
2. 成员取得相应训练等级或资格后，如长期未参加相关类型飞行，司令部可要求其进行复训或再评估。
3. 具体评定标准、考核方式和资格保持期限以训练部为准。
