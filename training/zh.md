## **第一条 适用范围**

1. 本训练大纲适用于虚拟中国空军（vPLAAF）全体飞行成员、教官成员及参与训练组织与评估的管理人员。
2. 本大纲适用于以下场景：
   - vPLAAF 内部基础、进阶及特种飞行训练；
   - vPLAAF 组织或参与的联合演练、专项训练与官方任务；
   - 教官评估、资格认定与复训活动。
3. 本大纲为训练性与组织性文件，不构成任何现实军事、政治或法律意义上的文件。

## **第二条 平台与规则合规声明**

1. vPLAAF 所有训练活动均在以下框架内进行：
   - VATSIM 网络规则（VATSIM Code of Conduct / VSOP / VATSIM Events Policy）
   - 与相关虚拟空管组织、VATSIM分部、vACC签署的LOA（Letter of Agreement）
   - vPLAAF 内部SOP、训练大纲与安全规定
2. 本大纲任何内容不得被解读为鼓励或模拟现实世界中的违法、危险或不当飞行行为。
3. 如本大纲与上级规则产生冲突，以上级规则为准。

## **第三条 术语与定义**

| 术语     | 定义                                           |
| -------- | ---------------------------------------------- |
| 学员     | 正在接受 vPLAAF 训练的飞行成员                 |
| 教官     | 经 vPLAAF 训练部认可，具备训练与评估资格的成员 |
| 阶段考核 | 针对某一训练阶段的综合性能力评估               |
| 合格     | 达到本大纲规定的最低安全与程序标准             |
| 不合格   | 未达到最低标准，需要补训或重新评估             |

## **第四条 训练组织架构**

1. vPLAAF 设立飞训作战部（Air Operations and Training），统一负责：
   - 训练体系设计与维护；
   - 教官资格管理；
   - 训练质量监督与标准更新。
2. 飞训作战部可下设：
   - 飞训作战部部长（Director of Operations）：负责监督、指导部门运行
   - 教官：负责执行教学任务、编写训练记录

## **第五条 职责划分**

**学员职责**

1. 遵守训练安排与教官指令；
2. 如实报告自身能力、设备状态与训练风险；
3. 在不适合继续训练时主动提出中止请求。

**教官职责**

1. 在安全、合规前提下组织训练；
2. 根据大纲进行客观评估，不得降低标准；
3. 在发现明显风险时，有权立即中止训练。

## **第六条 训练实施原则**

1. 所有训练应遵循：
   - 安全优先
   - 程序优先
   - 教学优先于通过
2. 不以“飞完时数”为唯一合格依据，能力达标为唯一标准。

## **第七条 训练记录**

1. 每次训练应至少记录：
   - 训练日期与时长；
   - 训练科目；
   - 教官姓名；
   - 主要问题与改进建议；
   - 是否计入阶段要求。
2. 训练记录采用：
   - 组织内部系统上传和管理文档；
   - 标准化训练表格；
3. 按vPLAAF隐私政策，学员本人可随时请求查阅本人所有训练记录

## **第八条 评估原则**

1. 所有评估应：
   - 以大纲条文为依据；
   - 以安全与程序为核心；
   - 避免主观偏好与非技术因素。
2. 单次不合格不等同于能力否定，应明确指出改进路径。

## **第九条 申诉与复评**

1. 学员对评估结果有异议的，可向飞训作战部提出复评申请；
2. 复评应由其他教官进行；
3. 复评结果为最终结果。

## **第十条 安全中止权**

1. 在以下情况下，教官或学员均有权中止训练：
   - 网络或设备异常；
   - 心理或身体状态不适；
2. 中止训练不计为失败。

## **第十一条 训练目标**

- 确保成员能安全规范飞行
- 掌握所需技能
- 熟练执行 SOP

## **第十二条 训练内容**

1. **VFR基础（不少于1小时）**

   - **训练目标**

     完成本阶段训练后，学员应能够：

     1. 理解影响飞机爬升、下降与能量管理的主要因素；
     2. 在目视条件下，按照标准程序建立并保持稳定的本场飞行；
     3. 正确执行起飞、初始爬升、平飞、下降、进近、复飞等阶段的操作流程；
     4. 在本场飞行中，能够根据需要选择并保持不同空速，且飞行姿态与配平合理；
     5. 在 ATC 存在或无 ATC 情况下，正确进行陆空通话或 CTAF 通话。

   - **主要内容**

     1. **地面与起飞阶段**
        - 起飞前检查单的规范使用；
        - 标准起飞程序与初始爬升；
        - 侧风起飞的基本修正方法（如适用）。
     2. **本场飞行（Circuits）**
        - 标准本场高度（通常 1000 ft AGL，按机场规定）；
        - 上风边、侧风边、顺风边、底边、最后进近的建立与保持；
        - 本场内爬升、下降阶段的姿态、功率与配平控制；
        - 不同空速下的爬升与下降控制；
        - 无襟翼 / 少襟翼本场（如机型适用）。
     3. **进近、复飞与异常情况**
        - 稳定进近的建立；
        - 标准复飞程序；
        - 模拟低高度复飞或不稳定进近中止；
        - 基本噪音控制与航迹管理意识。
     4. **通信与程序**
        - 受控机场下遵循 ATC 指令完成本场飞行；
        - 无 ATC 情况下的标准 CTAF 通话流程；
        - 正确理解并执行进近、起飞与本场相关指令。

   - **阶段要求**
     学员应在教官监督下完成不少于 1 小时的 VFR 基础飞行，并达到教官认可的最低操纵与程序标准，方可进入下一阶段。

   - **阶段考核标准**

     - 学员应能够在教官监督下，达到以下最低标准：
     - 本场高度保持在 **±100 ft** 以内；
     - 空速控制在目标值 **±10 KIAS** 以内；
     - 航向保持误差不超过 **±5°**；
     - 姿态控制平稳，无明显过度操纵；
     - 能在爬升、下降结束后，恢复稳定的平飞状态；
     - 正确使用配平，避免长期操纵力；
     - 能独立、安全完成至少 **3 个完整 VFR 本场**。

2. **VFR导航与夜航（不少于2小时）**

   - **训练目标**

     完成本阶段训练后，学员应能够：

     1. 在目视飞行条件下，完成短程航线的规划、执行与返场；
     2. 理解并正确使用基础导航手段进行航迹保持与定位；
     3. 在日间与夜间条件下，保持对高度、航向、空速与态势的有效控制；
     4. 在目视飞行中具备基本的地形、障碍物与交通避让意识；
     5. 在 ATC 存在或无 ATC 情况下，正确完成相关通话与程序。

   - **主要内容**

     1. **航前准备与航线规划**
        - 简单 VFR 航线规划（起降机场选择、备降考虑）；
        - 航向、距离、时间与油量的基础计算；
        - 飞行计划（Flight Plan）的基本填写与提交。
     2. **目视导航飞行**
        - 通过地标、航迹与目视参考进行导航；
        - 使用基础导航设备（如 VOR / DME / GPS 等，视机型与模拟条件）；
        - 航线飞行中的高度、航向与空速保持；
        - 航线偏差的识别与修正。
     3. **目视飞行中的态势感知**
        - 地形与障碍物避让原则；
        - 空域边界与受限空域的识别；
        - 交通观察与避让意识（“See and Avoid”）。
     4. **夜航基础**
        - 夜间视觉特性与空间定向风险；
        - 夜间起飞、爬升、进近与着陆的注意事项；
        - 机场灯光、跑道灯光与目视助航灯的识别与使用；
        - 夜间目视导航的限制与风险意识。
     5. **通信与程序**
        - 航线飞行中的 ATC 通话流程；
        - 无 ATC 情况下的标准通话与位置报告；
        - 正确理解并执行高度、航向与航线相关指令。

   - **阶段要求**
     学员应完成不少于 2 小时的导航训练（含日间与夜间科目），能够在教官提供一般性引导的前提下，独立完成一段简单航线并安全返场。

   - **阶段考核标准**

     学员应能够在教官监督下，达到以下最低标准：

     - 能正确识别并使用至少一种基础导航台或导航手段（如 VOR / DME / GPS），用于航迹保持与位置确认；
     - 能正确识别并调谐指定的 VOR / DME 导航台，并理解其径向与距离信息；
     - 能建立并保持指定的 VOR 径向或 VOR-DME 弧线，航迹偏差可控且修正及时；
     - 能通过目视参考或基础导航手段维持合理航迹；
     - 能通过导航台信息与目视参考相结合，维持合理的态势感知与航迹判断；
     - 夜航阶段操作平稳，无明显空间定向错误；
     - 能安全、独立完成至少一次完整 VFR 航线飞行（含起飞与着陆）。

3. **IFR 程序与仪表飞行训练（不少于 3 小时）**

   - **训练目标**

     完成本阶段训练后，学员应能够：

     1. 在仪表气象条件下，依靠仪表安全控制飞机姿态与飞行轨迹；
     2. 理解并正确执行标准仪表飞行程序，包括起飞、离场与进场程序、航路、进近与复飞；
     3. 正确使用常见仪表进近与导航手段完成受控飞行；
     4. 在 ATC 指挥下，准确理解并执行高度、航向、速度与航路相关指令；
     5. 在高工作负荷环境下，保持基本的态势感知与程序一致性

   - **主要内容**

     1. **仪表飞行基础**
        - 基本飞行仪表的识别与使用；
        - 姿态、功率与性能的仪表化控制；
        - 基于仪表的爬升、平飞与下降；
        - 标准转弯、改平与航向保持。
     2. **标准仪表程序（SID / STAR）**
        - 标准仪表起飞程序（SID）的理解与执行；
        - 标准仪表进场程序（STAR）的理解与执行；
        - 高度、速度限制的识别与遵守；
        - 程序偏差的识别与修正。
     3. **航路与导航**
        - 基于 VOR / DME / RNAV 的航路飞行；
        - 航路段高度管理与航迹保持；
        - 航路中 ATC 指令的执行（雷达引导、直飞、改航、偏置等）；
        - 航路态势感知与工作负荷管理。
     4. **仪表进近**
        - 精密进近与非精密进近的基本概念；
        - ILS / LOC / VOR / NDB / RNAV（LNAV）等进近程序的执行；
        - 进近稳定性判断与持续监控；
        - 最低高度 / 决断高度的理解与遵守。
     5. **等待与复飞**
        - 等待航线（Holding）的建立、进入与保持；
        - 标准复飞程序的理解与执行；
        - 不稳定进近的中止与改飞；
        - 复飞后程序与 ATC 协调。
     6. **夜间 IFR 与复杂情景**
        - 夜间 IFR 起飞与进近注意事项；
        - 多重指令下的程序执行；
        - 模拟设备或系统异常下的基本处置（在教官监控下）。

   - **阶段要求**
     学员应在线完成不少于 3 小时的 IFR 训练，并在至少一次完整 IFR 航线中，展示对基础程序和 ATC指令的正确理解与执行，经教官评定合格后方可进入第四阶段。

   - **阶段考核标准**

     学员应能够在教官监督下，达到以下最低标准：

     - 能正确理解并执行指定的 SID、STAR、航路及进近程序，遵守相关高度、速度与航迹限制；
     - 能使用 VOR / DME / RNAV 等仪表导航手段保持指定航迹，并进行合理、及时的修正；
     - 能独立完成至少一种仪表进近（ILS / LOC / VOR / RNAV），并正确理解与遵守 MDA / DA；
     - 能在需要时立即、正确执行标准复飞程序，并完成复飞后的程序衔接；
     - 能建立并保持等待航线，理解进入方式并保持航迹稳定；
     - 能准确理解、复诵并执行 ATC 的高度、航向、速度及程序指令；
     - 能在较高工作负荷环境下维持程序优先级判断，避免明显程序性错误；
     - 能安全、完整地完成至少一次 IFR 航线飞行（起飞—航路—进近—着陆或复飞）。

4. **特种作业与机型专项训练**

- **训练目标**

  完成本阶段训练后，飞行员应能够：

  1. 理解并执行 vPLAAF 虚拟特种作业相关的基本战术与程序；
  2. 熟练掌握所飞行机型的专项操作流程与性能特性；
  3. 在符合 VATSIM 与 VSOP 规则的前提下，安全执行低空、编队及特定任务科目；
  4. 在多机协同或复杂任务环境下，保持程序一致性、通信规范与态势感知；
  5. 具备参与 vPLAAF 正式任务或联合演练的基本能力

- **主要内容**（可根据机型与任务类型选取若干科目组合完成）

  1. **机型专项操作**
     - 机型启动、关车与特殊系统操作流程；
     - 机型性能限制、包线理解与能量管理；
     - 常见异常与应急情况的模拟处置（如发动机异常、系统失效等）。
  2. **低空飞行与航迹管理**
     - 低空飞行的高度、速度与风险控制原则；
     - 地形跟随与地形规避意识；
     - 在允许空域内进行的低空航迹规划与保持；
     - 低空飞行中与 ATC 或任务指挥的协调。
  3. **编队飞行**
     - 基本编队队形（如双机、三机或四机）；
     - 编队集合与解散程序；
     - 编队内位置保持、相对运动与安全间隔；
     - 编队飞行中的通信分工与职责划分。
  4. **特种任务科目**
     - 空运、空投与补给任务流程；
     - 护航、伴飞与编组转场任务；
     - 任务空域内的航迹保持与冲突规避；
     - 任务中与其他虚拟组织的协同（如适用）。
  5. **高级科目（视条件选取）**
     - 空中加油程序；
     - 基础空战机动（能量管理、进攻/防御机动等，仅限允许空域）；
     - 特殊使用空域（SUA / MOA / Military Training Routes）的理解与模拟使用。
  6. **任务流程与风险管理**
     - 任务前简报（Briefing）的参与与理解；
     - 任务执行中的程序遵循与变更应对；
     - 任务后讲评（Debriefing）与经验总结；
     - 网络安全、他人体验与合规意识。

- **阶段要求**

  1. 飞行员应完成不少于 4 小时的高级特种作业训练。
  2. 通过由教官或标准与评估组组织的综合性实践考试，验证其在任务规划、程序执行、通信和飞行安全等方面达到 vPLAAF 规定标准。
  3. 通过本阶段考核后，飞行员方可被认定为具备参加 vPLAAF 组织的正式任务/联合演练之资格。

## **第十三条 资格评定与保持**

1. 各训练阶段结束后，需通过理论测试、模拟任务及实践考核等方式进行评定。
2. 成员取得相应训练等级或资格后，如长期未参加相关类型飞行，司令部可要求其进行复训或再评估。
3. 具体评定标准、考核方式和资格保持期限以训练部为准。

## **第十四条 未尽事宜**

对本训练大纲未予明确规定的情况，由 vPLAAF 飞训作战部在 vPLAAF 司令部指导下根据 VATSIM 相关规则、本组织宗旨及公平合理原则作出处理，并视需要补充或修订相关条款。
